    O Que é Git e GitHub?

    Git é um sistema de controle de versão que permite rastrear e gerenciar alterações em projetos de software. GitHub é uma plataforma online que hospeda repositórios Git, permitindo colaboração e compartilhamento de código.

    Configurando uma Chave SSH

    1. Gerar uma chave SSH usando o comando: ssh-keygen -t rsa -b 4096 -C "seu-email@example.com"
    2. Adicionar a chave SSH ao agente: ssh-add ~/.ssh/id_rsa
    3. Copiar a chave pública: clip ~/.ssh/id_rsa.pub (Windows) ou pbcopy ~/.ssh/id_rsa.pub (macOS)
    4. No GitHub, vá em "Settings" > "SSH and GPG keys" > "New SSH key" e cole a chave pública.

    Criando e Clonando um Repositório

    1. No GitHub, clique em "New" para criar um novo repositório.
    2. No seu terminal, vá para o diretório onde deseja clonar o repositório: cd caminho/do/diretorio
    3. Clone o repositório usando o comando: git clone URL_DO_REPOSITORIO
    4. Entre no diretório do repositório: cd nome-do-repositorio
    5. Agora você pode fazer alterações nos arquivos, adicionar, commitar e fazer push para o GitHub.
